# Starstruck Growth Campaign (Ethical)

Goal: increase organic stars through quality, discoverability, and consistency.

## 1) Product clarity
- Keep README first 15 lines crystal-clear (what, who, why, quickstart)
- Add one GIF/screenshot showing the key workflow
- Keep changelog/release notes current

## 2) Discoverability
- Maintain accurate GitHub topics
- Keep repo description and homepage current
- Publish small but regular releases

## 3) Social proof
- Use Issues + Discussions for roadmap transparency
- Label beginner-friendly tasks (`good first issue`)
- Add CONTRIBUTING and reproducible local setup

## 4) Distribution (no spam)
- Share meaningful updates in relevant communities
- Post release notes with before/after value
- Ask for feedback, not stars

## Weekly KPI
- New stars/week
- New contributors/week
- New issues/discussions with actionable feedback

## Current execution checklist
- [x] CONTRIBUTING.md exists
- [x] v0.1.0 release exists
- [x] Core topics configured
- [ ] Publish v0.1.1 with UX reliability improvements
- [ ] Open a "good first issue" item
- [ ] Post one concise update with screenshot

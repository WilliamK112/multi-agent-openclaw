# Contributing

Thanks for checking out **multi-agent-openclaw**.

## Quick ways to contribute
- Report reproducible bugs with steps + screenshots
- Improve docs and examples
- Suggest workflow templates for specific tasks
- Add focused tests for orchestrator behavior

## Local setup
```bash
npm install
npm run dev:server
```

Open: `http://127.0.0.1:8787`

## PR guidelines
- Keep PRs small and focused
- Include before/after behavior in the PR body
- For UI changes, attach screenshots
- Run tests before opening PR

## Code of conduct
Be respectful, constructive, and specific.
